### Usage
詳細模式開關可切換書籍詳細資訊顯示與否。

優惠價開關可操控各書目顯示折扣價格或原價。

大/小呈現開關可切換顯示內容大小。





陳列資料中的書籍資訊與圖像，可自由切換詳細/簡略瀏覽模式，切換特價狀態與否，並顯示個別金額。
